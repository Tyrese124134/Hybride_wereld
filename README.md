# Leuvehaven Immersive Experience
Projectoverzicht
Een interactieve historische museumervaring waarin de geschiedenis van de Leuvehaven digitaal tot leven wordt gebracht.
Het project combineert:
Unreal Engine omgeving
Historische 3D reconstructie
Cinematische storytelling
AI contentmoderatie
Bezoekersinteractie
Het doel is om bezoekers niet alleen geschiedenis te laten bekijken, maar onderdeel te laten worden van de ervaring.


⸻


## Onderdelen
Unreal Engine Experience
Een digitale stijlkamer gebouwd in Unreal Engine.
Bevat:
Blender 3D modellen
Cinematic camera systeem
Audio omgeving
Fire VFX
Image streaming systeem
Documentatie:
/Unreal-Experience


⸻


## AI Drawing Moderation
Een AI-systeem dat bezoekersbijdragen controleert voordat deze worden weergegeven.
Bevat:
PyTorch ResNet18 model
FastAPI backend
Feedback learning
Safe/Unsafe detectie
Documentatie:
/AI-Drawing-Moderation
